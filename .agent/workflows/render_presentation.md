# Render Quarto Presentation

To compile and preview the presentation, run the appropriate rendering commands.

1. Render the presentation to HTML (Revealjs):
```bash
quarto render present/presentation.qmd --to revealjs
```

2. Render the presentation to PowerPoint (PPTX):
```bash
quarto render present/presentation.qmd --to pptx
```
